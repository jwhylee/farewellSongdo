# farewellSongdo

This is a text-based RPG game.
created by Jaeyeol Lee for Course GAI2008

## Features

- Move around the campus
- Buy items
- Use items
- Check status
- Check bag
- Quit the game

## How to Play

- w, a, s, d to move
- v to print status
- b to open bag
- q to quit game
