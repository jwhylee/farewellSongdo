# farewellSongdo

### This is a text-based RPG game.  
Game Language : Korean

*Created by Jaeyeol Lee for Course GAI2008*

## Features

- **Campus Exploration**: Navigate through an extensive grid-based map of Yonsei University's Sinchon Campus.
- **Quest System**: Uncover campus mysteries! Receive quests, investigate incidents, and solve puzzles to complete the game.
- **Economic System**: Buy recovery items from specific shops and sell them at varying prices across the campus to earn profit.
- **Inventory & Status Management**: Use items to restore HP, and keep track of your health, money, and current location.
- **Multiple UI Modes**: Play in "Full" mode for an immersive visual layout with a mini-map, or "Minimal" mode for a streamlined text experience.
- **Difficulty Selection**: Choose between "Normal" and "Hard" to adjust how quickly your HP depletes when moving.
- **Save & Load**: Save your game progress and resume your adventure at any time.

## How to Play

Enter the following Korean commands into the prompt to play the game:

### Movement
- `동`, `서`, `남`, `북`: Move one step in the specified direction (East, West, South, North).

### Interaction
- `구매`: Buy items at designated shop locations.
- `판매`: Sell items at designated selling spots to earn money.
- `임무`: Interact with quest-related NPCs or locations to receive or solve quests.

### Status & Inventory
- `가방`: Open your bag to check items and consume them for HP recovery.
- `상태`: Check your current HP, money, location, and valid adjacent paths.
- `임무목록`: Check the list of active quests and their descriptions.

### System Options
- `난이도`: Check or change the current game difficulty.
- `저장`: Save the current game state to a file.
- `불러오기`: Load a previously saved game file.
- `도움말`: View the list of all available commands.
- `닫기`: Close the current popup window (e.g., bag, status panel).
- `종료`: Exit the game.
