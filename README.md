# farewellSongdo

### This is a text-based RPG game.  
Game Language : Korean

*Created by Jaeyeol Lee for Course GAI2008*

## Features

- Move around the campus
- Buy items
- Use items
- Check status
- Check bag
- Save and Load game
- Quit the game

## How to Play

- w, a, s, d to move
- v to print status
- b to open bag
- 1 to save game
- 2 to load game
- q to quit game
